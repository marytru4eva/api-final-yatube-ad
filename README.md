# Yatube API

API для социальной сети Yatube. Позволяет создавать посты, комментарии, группы и подписки.

## Возможности

- Регистрация и аутентификация через JWT-токены
- Создание, редактирование и удаление постов
- Добавление комментариев к постам
- Просмотр групп
- Подписка на других пользователей
- Поиск по подпискам

## Технологии

- Python 3.9
- Django 3.2
- Django REST Framework 3.12
- JWT-аутентификация
- SQLite (по умолчанию)
